## Project Overview

AgriMitra is an AI-powered farmer assistance system developed to address critical agricultural challenges such as crop disease identification, market price fluctuations, and limited access to expert guidance. The platform combines Deep Learning, Machine Learning, and Voice-Based Interaction to provide farmers with real-time insights and actionable recommendations.

The system enables farmers to upload tomato leaf images for disease detection, access mandi price predictions, receive personalized farming advice, and interact with the platform using voice commands in regional languages.

---

## Key Features

### Crop Disease Detection

* Detects tomato leaf diseases using a CNN-based MobileNetV2 model.
* Supports early disease identification and preventive action.
* Provides accurate classification of healthy and infected leaves.
* Achieves high prediction accuracy using transfer learning techniques.

### Mandi Price Prediction

* Predicts crop prices using Machine Learning models trained on historical market data.
* Provides real-time market insights and trend analysis.
* Helps farmers make informed selling decisions.

### Voice-Based Farmer Assistance

* Supports multilingual voice interaction using Web Speech API.
* Converts speech into text for seamless communication.
* Improves accessibility for farmers with limited digital literacy.

### Personalized Farming Recommendations

* Generates crop-specific recommendations based on detected diseases and market trends.
* Provides guidance on pesticides, fertilizers, irrigation, and crop management.
* Offers actionable insights for improving productivity.

### User-Friendly Dashboard

* Simple and responsive web interface.
* Easy navigation for all system modules.
* Real-time display of predictions and recommendations.

## Dataset

The crop disease detection model is trained using the PlantVillage Tomato Leaf Dataset containing healthy and diseased tomato leaf images across multiple disease categories.

Disease Classes Include:

* Healthy
* Early Blight
* Late Blight
* Leaf Mold
* Septoria Leaf Spot
* Bacterial Spot
* Target Spot
* Mosaic Virus
* Yellow Leaf Curl Virus
